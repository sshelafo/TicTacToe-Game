# TicTacToe Game

## Overview
A command-line TicTacToe game implementation in Java.

## Features
- Player vs Player gameplay
- Input validation
- Win detection for all conditions
- Clean, formatted board display

## Setup
1. Compile: `javac src/*.java`
2. Run: `java TicTacToe`

## How to Play
- Players alternate turns
- Enter row and column (1-3) to place your mark
- First to get 3 in a row wins!

## Project Structure
- src/ - Source code files
- tests/ - Unit test files
- docs/ - Documentation
- config.properties - Game configuration
